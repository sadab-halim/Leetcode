## [Peak Index in Mountain Array](https://leetcode.com/problems/peak-index-in-a-mountain-array/submissions/)

*Example-1:*
```
Input: arr = [0,1,0] 
Output: 1
```

*Example-2:*
```
Input: arr = [0,2,1,0]
Output: 1
```

*Example-3:*
```
Input: arr = [0,10,5,2]
Output: 1
```

*Constraints:*

3 <= arr.length <= 104 <br/>
0 <= arr[i] <= 106 <br/>
arr is guaranteed to be a mountain array. <br/>
